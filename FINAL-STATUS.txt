🚀 推荐方案（最快）
════════════════════════════════════════════════════

方案 A: Git Bundle（3 步）

   1. 下载 bundle 到本地
      scp root@server-ip:/root/.openclaw/workspace/crypto-dashboard/crypto-dashboard.bundle ./

   2. 本地克隆并推送到 GitHub
      git clone crypto-dashboard.bundle crypto-dashboard
      cd crypto-dashboard
      export GITHUB_TOKEN=your_token
      git remote add origin https://$GITHUB_TOKEN@github.com/YOUR_USERNAME/crypto-dashboard.git
      git push -u origin main

   3. 连接 Cloudflare Pages（浏览器操作）
      https://dash.cloudflare.com/ → Workers & Pages → Create → Connect to Git
      选择仓库 → 配置 build → 完成

方案 B: 自动部署脚本（在服务器执行）

   1. 生成 GitHub Token（https://github.com/settings/tokens）

   2. 执行脚本
      cd /root/.openclaw/workspace/crypto-dashboard
      ./auto-deploy.sh

   3. 输入用户名和 Token，自动推送
